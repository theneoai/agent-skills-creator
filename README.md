# Agent Skills Creator

A tool for creating, evaluating, and optimizing AI agent skills.

## Structure

- `SKILL.md` - Main skill definition
- `scripts/` - Scripts for skill management
  - `skill-manager/` - Core skill management tools
- `references/` - Documentation and templates
- `evals/` - Evaluation data
- `test_cases/` - Test cases
- `test_results/` - Test results

## Usage

See `SKILL.md` for detailed usage instructions.
